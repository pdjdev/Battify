## 주요 기능

- 작업 표시줄에서 확인 가능한 배터리 퍼센트 아이콘
- 충전 상태 변화시 알림 팝업
- Windows 10/11의 라이트/다크 테마 맞춤 스타일
- HiDPI 지원 GUI

## 요구 사양
- Windows 10+ 이상 버전
- .NET 8
